# ${\color{red} Banking\ Analytics\ Dashboard}$

# ✅ ${\color{green} Objective}$
The objective of this project is to analyze banking customer data by integrating Excel, MySQL, Python, and Power BI to develop an interactive business intelligence dashboard. The project helps visualize customer, loan, and deposit information, enabling data-driven decision-making through meaningful insights and dynamic reports.

# 🛠️ ${\color{grey} Tools\ and\ Technologies\ Used}$
- **Microsoft Excel** – Raw data collection and organization
- **MySQL** – Database creation, storage, and SQL querying
- **Python** – Data extraction, preprocessing, and exploratory data analysis
- **Pandas & NumPy** – Data cleaning and transformation
- **MySQL Connector** – Database connectivity
- **Power BI Desktop** – Dashboard development and visualization
- **Power Query** – Data transformation and preparation
- **DAX (Data Analysis Expressions)** – KPI and measure creation

# 🔄 ${\color{blue} Steps\ Included\ /Implementation\ Flow}$
1. Collected the banking dataset in Excel.
2. Imported the dataset into MySQL and created the database.
3. Queried banking data using SQL.
4. Connected Python to MySQL and loaded data into Pandas.
5. Cleaned and transformed the dataset using Python.
6. Imported the processed data into Power BI.
7. Created data relationships and DAX measures.
8. Developed interactive dashboards for Home, Loan Analysis, Deposit Analysis, and Summary.

# 📊 ${\color{orange} Analysis\ and \ Observations}$
- Analyzed customer demographics and banking relationships.
- Evaluated loan and deposit performance across customer segments.
- Compared business lending, savings, and account balances.
- Monitored KPIs such as Total Clients, Total Loans, and Total Deposits.
- Used interactive filters and charts to identify banking trends and customer behavior.

# 🔍 ${\color{blue} Key\ Insights \/Learnings}$
- Learned the complete end-to-end data analytics workflow from data collection to visualization.
- Gained hands-on experience with SQL, Python, and Power BI integration.
- Improved skills in data cleaning, transformation, and business intelligence reporting.
- Developed dynamic dashboards using DAX measures and interactive visuals.
- Enhanced understanding of banking analytics and data-driven decision-making.

# 🧾 ${\color{black} Conclusion}$
This project demonstrates an end-to-end banking analytics solution by integrating Excel, MySQL, Python, and Power BI. It transforms raw banking data into interactive dashboards that provide valuable business insights, enabling efficient analysis of customer, loan, and deposit performance while supporting informed decision-making.
